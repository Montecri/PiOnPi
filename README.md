Infinitelly Calculating Digits of Pi on Rapsberry Pi Pico

| MAX7219 Module |      Pico     |
|----------------|---------------|
| VCC            | 3V3  (pin 36) |
| GND            | GND  (pin 38) |
| DIN            | GP19 (pin 25) |
| CS             | GP17 (pin 22) |
| CLK            | GP18 (pin 24) |

Makes use of:
- https://github.com/ozhantr/DigitLedDisplay/

C code from:
- https://bellard.org/pi/

Adapted to this use.
